This is a public installation of GNUkebox, as part of the Libre.fm project. GNUkebox implements a basic server for storing listening data from various music applications, these are also known as scrobbles. GNUkebox is part of GNU FM, and developed for Libre.fm by FooCorp and contributors.

GNUkebox implements little more than this page. You can even turn this page off.

If you'd like to use our server (everyone is welcome!), please visit Libre.fm
